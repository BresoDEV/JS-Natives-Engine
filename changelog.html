V1.0 - 20/07/2024

Lançamento da versao beta

V1.01 - 21/07/2024

Adições:

GET_ENTITY_COLLIDED: Retorna o objeto que outro objeto colidiu

SHOOT_ENTITY: Cria um projetil que vai em direção de um objeto

CREATE_SHOOT_TO_COORD: Cria um projetil que vai em direção de uma coordenada

MAKE_ENTITY_FOLLOW_PLAYER: Faz com que um elemento siga ou vá na direção do jogador

MAKE_ENTITY_FOLLOW_ENTITY: Faz com que um elemento siga ou vá na direção de outro elemento

DESTROY_ENTITY_WHEN_COLLIDE_TO_ANY_ENTITY: Faz com que um elemento se auto-destrua ao colidir com qualquer bloco com colisão (não necessita ser chamada em loop)

GET_MAP_IMG: Retorna o caminho da imagem do mapa atual

SET_MAP_IMG: Define a imagem do mapa atual

DRAW_MINIMAP: Exibe/oculta a miniatura do mapa atual atraves de uma tecla qualquer

ADD_KEYBOARD_EVENT: Executa uma função atraves de uma tecla qualquer

GET_ALL_ENTITYS_WITH_COLLISION: Retorna um array com todos elementos do mapa que possue colisão

SET_AS_ENEMY: Define um elemento como inimigo, podendo perseguir ou não o jogador

SET_AS_FRIEND: Define um elemento como amigo

GET_ALL_ENEMYS: Retorna um array com todos elementos do mapa que são inimigos

GET_ALL_FRIENDS: Retorna um array com todos elementos do mapa que são amigos

SET_ENTITY_NO_LONGER_NEEDED: Deleta um elemento permanentemente ao sair do campo de visão

GET_DISTANCE_BETWEEN_ENTITYS: Calcula a distancia entre 2 elementos

GET_DISTANCE_BETWEEN_COORDS: Calcula a distancia entre 2 pontos

ATTACH_AUDIO_TO_ENTITY: Linka um audio a um elemento(quanto mais perto do player, mais alto), dando a sensação que o audio esta tocando no exato local do mapa

DRAW_LIGHT: Cria uma luz artificial em um ponto do mapa

SET_LIGHT_SIZE: Muda o tamanho de uma luz artificial

SET_LIGHT_COLOR: Muda a cor de uma luz artificial

SET_PROGRESS_BAR_COLOR: Muda a cor de uma progressbar

SET_PROGRESS_BAR_MAX_VALUE: Muda o valor maximo de uma progressbar

SET_PROGRESS_BAR_MIN_VALUE: Muda o valor minimo de uma progressbar

HIDE_MOUSE: Oculta o cursor do mouse

SHOW_MOUSE: Exibe o cursor do mouse

ADD_MOBILE_BUTTON: Adiciona botoes extras para smartphones

SET_MOBILE_BUTTON_TEXT: Muda o texto de botoes extras para smartphones

SET_MOBILE_BUTTON_BG_COLOR: Muda a cor de fundo de botoes extras para smartphones

SET_MOBILE_BUTTON_FONT_COLOR: Muda a cor da fonte de botoes extras para smartphones

DELETE_ALL_MOBILE_BUTTONS: Deleta todos botoes extras para smartphones

GET_CLOSEST_FRIEND: Retorna o elemento amigo mais próximo do player

GET_CLOSEST_ENEMY: Retorna o elemento inimigo mais próximo do player

Correções:

GET_ENTITY_ROTATION: Retorna um NUMBER, e não uma STRING, podendo manipular-lo com operações matemáticas

DOES_ENTITY_EXIST: Retorna corretamente, via objeto ou via id

DRAW_MINIMAP: Adaptado para smartphones

PLAY_AUDIO: Inicia sempre depois da interação. Antes não executava

V1.03 - 25/07/2024

Adições:

SET_GAME_NAME: Define o nome do jogo, utilizado em opções de saves

GET_GAME_NAME: Obtem o nome do jogo, utilizado em opções de saves

DOES_GAME_NAME_AS_DEFINED: Verifica se o nome do jogo ja foi definido

SAVE_INVENTORY_ITEM_QUANT: Salva a quantidade de um item no sistema de inventário

ADD_INVENTORY_ITEM_QUANT: Adiciona uma quantidade a mais, de um item no sistema de inventário

REMOVE_INVENTORY_ITEM_QUANT: Remove uma quantidade de um item no sistema de inventário

PLAYER_HAVE_ITEM: Verifica se o player tem uma quantidade de um item no sistema de inventário

GET_INVENTORY_ITEM_QUANT: Retorna a quantidade de um item no sistema de inventário

SAVE_INVENTORY_ITEM_IMG: Salva uma imagem em um item no sistema de inventário

GET_INVENTORY_ITEM_IMG: Retorna a imagem de um item no sistema de inventário

CLEAR_ALL_INVENTORY: Zera todas quantidades de itens do sistema de inventário

SET_ENTITY_VISIBLE: Define se um elemento fica visível ou não

SET_CAMERA_FOCUS_ON_COORD: Define a posição da camera a partir de uma coordenada

RESTORE_CAMERA_FOCUS_ON_PLAYER: Re-define a posição da camera no player

GET_TOUCH_X_COORD: Retorna a posição X do touch da tela em smartphones

GET_TOUCH_Y_COORD: Retorna a posição Y do touch da tela em smartphones

ADD_VOID_TOUCHED_AT_AREA: Executa algo quando o touch da tela for em uma area específica

ADD_VOID_TOUCHED_TO_ENTITY: Executa algo quando o touch da tela for em um elemento

ADD_VOID_CLICK_TO_ENTITY: Executa algo quando der um click em um elemento

Correções:

GET_ENTITY_Y_COORD_ON_SCREEN: Retornando valor incorreto

GET_ENTITY_CENTER_Y_COORD_ON_SCREEN: Retornando valor incorreto por causa do nativo acima
